# Code for Osnabrück
<div class="flex justify-center">
    <Image src={logo} width={300} alt="Logo Code for Osnabrück" />
</div>

**Code for Osnabrück** ist eine lokale Initiative von engagierten Bürger:innen, die gemeinsam an digitalen Projekten für die Stadt arbeiten. 
Sie gehört zum Netzwerk von **Code for Germany** und entwickelt offene Datenanwendungen, Apps und Tools für mehr Transparenz und Beteiligung. 
Ziel ist es, mithilfe von Technologie das Leben in Osnabrück zu verbessern und städtische Probleme kreativ zu lösen. Solche Projekte sind wichtig, 
weil sie Bürgerbeteiligung stärken und Innovation im öffentlichen Raum fördern.

## Aktuelle Projekte

- **RSS-Bots für Mastodon**  
  Entwicklung von Bots, die Inhalte aus verschiedenen RSS-Feeds automatisch auf Osna.social veröffentlichen. 
  Darunter fallen unter anderem Informationen aus dem Ratsinformationssystem der Stadt Osnabrück.

- **Monitoring von Parkhausdaten**  
  Analyse und Auswertung von Parkhausdaten zur besseren Darstellung der Auslastung von Parkhäusern – insbesondere in den Nachtstunden. 
  Ziel ist es, auf dieser Grundlage zu prüfen, ob Parkhäuser als Quartiersgaragen genutzt werden können.
